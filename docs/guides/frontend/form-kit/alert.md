# Alert

Displays an alert in the form.

## @icon

An optional icon to use in the alert.

**Examples**

```hbs
<form.Alert @icon="info-circle">
  Foo
</form.Alert>
```

## @type

An optional icon to use in the alert. Allowed types: `success`, `error`, `warning` or `info`.

**Examples**

```hbs
<form.Alert @type="warning">
  Foo
</form.Alert>
```
